# airfreight-gp-dashboard
## Overview 
This powerbi  dashboard was designed to enhance visibility into operational activities that influence financial performance across the MEA region. Its core logic is to highlight files from previous periods that negatively affect the financial results of the month currently under review. 
## Business Problem
Financial and operational teams needed better visibility into why GP was changing across files and months, especially where repeated file touches were affecting performance.

## Objective
To build a dashboard that helps decision-makers monitor GP variance, identify sick files, and flag operational issues with financial impact.

## Tools Used
- Power BI
- Excel
- Power Query
- DAX

## Key Features
- GP variance tracking
- File-level performance visibility
- Dashboard views for weekly monitoring

## My Role
I worked on the dashboard logic, structure, calculations, and business-facing presentation of the insights to support regional decision-making.

## Key Insights
- Certain files were repeatedly adjusted across months
- Delayed billing affected visibility of actual GP performance
- High-risk files could be flagged earlier through better monitoring
